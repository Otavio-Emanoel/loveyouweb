# 💖 LoveYouWeb 💖

> *"Um cantinho especial na internet, criado com todo o amor do mundo para a Ágata!"*

Este é um projeto de login interativo e página de boas-vindas personalizado, desenvolvido com muito carinho para surpreender o amor da minha vida, a **Ágata**. O site é repleto de animações fofas, transições românticas e detalhes interativos pensados exclusivamente para fazê-la sorrir!

---

## 🌸 Detalhes Fofos & Funcionalidades

### 1. 🐰 Mascote Interativo (Coelhinho)
No topo do formulário de login está um coelhinho feito em SVG e React que reage de forma muito fofa a cada ação:
*   **Olhar Curioso**: Quando você digita no campo de e-mail, os olhinhos dele se movem para baixo para ver o que você está escrevendo.
*   **Esconde-Esconde (Peek-a-boo)**: Quando você clica no campo de senha, ele levanta as patinhas e cobre os olhos com vergonha.
*   **Espiadinha**: Se você clicar no botão de "mostrar senha" (o ícone de olho), ele abaixa uma das patinhas para dar uma espiadinha tímida com um olho só!
*   **Felicidade & Vergonha**: Ao passar o mouse por cima do botão de entrar, o coelhinho fica muito feliz, dá um sorriso fofo mostrando a linguinha e as bochechas dele coram em rosa brilhante.

### 2. ✨ Elementos Mágicos na Tela
O fundo do site é composto por um gradiente suave em tons pastel de rosa e lavanda, com várias animações delicadas flutuando:
*   **Nuvens fofas** flutuando lentamente pelo céu.
*   **Estrelas brilhantes** cintilando em diferentes velocidades.
*   **Corações flutuantes** subindo suavemente da base da tela.

### 3. 💌 Dica Secreta ("Esqueceu a senha?")
Se a Ágata esquecer a senha e clicar em "Esqueceu a chave secreta?", um lindo pop-up modal personalizado aparece com uma dica romântica:
> *"A senha é o nome da pessoa que você mais ama no mundo inteiro!"* 💖

### 4. 🚀 Tela de Carregamento com Barra de Progresso
Ao colocar o login correto, o site exibe uma barra de progresso romântica que carrega de 0% a 100% levando um coraçãozinho pulsante com ela, acompanhado de mensagens carinhosas:
1.  *“Empacotando abraços virtuais...”* 📦
2.  *“Reunindo estrelas brilhantes...”* ✨
3.  *“Abrindo o seu lugar feliz...”* 🌸
4.  *“Entrando... Segure firme!”* 💖

### 5. 🏡 Página de Boas-vindas & Carta de Amor
Após o carregamento da barra, o site redireciona para a página `/welcome`, onde a Ágata é recebida com:
*   Um cartão de boas-vindas personalizado com o nome dela.
*   O mascote coelhinho comemorando alegremente (acenando e mexendo as orelhinhas).
*   Um botão interativo **"Abrir Surpresa"** que, ao ser clicado, revela uma linda carta de amor escrita à mão em um papel estilo vintage.

## 🛠️ Tecnologias Utilizadas

*   **Next.js 16 (App Router)** para estrutura de rotas e renderização limpa.
*   **TypeScript** garantindo tipagem sólida.
*   **TailwindCSS v4** com estilos utilitários premium.
*   **CSS Keyframes nativos** para animações super leves, fluidas e compatíveis.

## 🚀 Como Executar o Projeto Localmente

1.  Clone ou navegue até a pasta do projeto.
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
4.  Abra o navegador no endereço: [http://localhost:3000](http://localhost:3000)
